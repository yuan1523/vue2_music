介绍
该项目是基于 Vue 和 ElementUI 组件库 搭建的一个PC端的音乐网站，能登录网易云音乐用户账号，听歌、歌词滚动等功能

技术栈
Vue3 / Vue-router / Vant / Axios / Vuex
